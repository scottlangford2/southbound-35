# 254 Counties, One Interstate: The Hays County Growth Story

Replication code for the [blog post](https://scottlangford2.github.io/scott_langford/posts/2026/04/hays-county-growth/).

## Quickstart

```bash
pip install -r requirements.txt
python build_figures.py
```

Figures are written to `figures/`.

## Figures

| Figure | File | Description |
|--------|------|-------------|
| 1 | `hays_population.png` | Hays County population, 1990–2025 |
| 2 | `hays_cities.png` | City-level comparison: 2010 vs. 2025 |
| 3 | `hays_affordability.png` | Median home prices: Travis vs. Williamson vs. Hays |

## Data Sources

- U.S. Census Bureau (decennial census 1990–2020, ACS 2023)
- City of Kyle Economic Development
- City of San Marcos population estimates
- Redfin, ABoR/CBA Realtors (home prices, February 2026)
