{%- comment -%} giscus — GitHub Discussions as blog comments. No database, no tracking, commenters sign in with GitHub. Routed via the theme's "custom" comments provider slot so no upstream theme files have to be modified. Config is read from _config.yml under `giscus:`. Setup (one-time, on the blog repo): 1. Enable GitHub Discussions: repo → Settings → General → Features → check "Discussions". 2. Install the giscus GitHub App on the repo: https://github.com/apps/giscus 3. Generate config values at https://giscus.app — it prints the repo_id and category_id for your chosen Discussion category. Paste those into _config.yml under `giscus:`. 4. In _config.yml set comments.provider: custom to route here. {%- endcomment -%} {%- if site.giscus.repo -%} {%- endif -%}
